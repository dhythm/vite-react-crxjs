# vite-react-crxjs

## How to create the environment

```sh
npm init vite@latest
✔ Project name: … vite-react-crxjs
✔ Select a framework: › React
✔ Select a variant: › TypeScript

cd vite-react-crxjs
npm install
npm i -D @crxjs/vite-plugin @types/chrome --legacy-peer-deps
```

## Getting Started

```sh
npm install
```
